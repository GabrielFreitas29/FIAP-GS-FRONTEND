# 🌎 SentinelaOrbital

Landing Page desenvolvida para apresentar o projeto SentinelaOrbital.

## 📋 Sobre

O SentinelaOrbital é uma solução de monitoramento climático que utiliza dados de satélites e sensores IoT para auxiliar na prevenção de enchentes e queimadas.

## 🚀 Tecnologias

- HTML5
- CSS3
- Flexbox
- Google Fonts
- Font Awesome

## 🎯 Objetivo

Fornecer alertas preventivos para ajudar comunidades vulneráveis a se prepararem para desastres climáticos.
